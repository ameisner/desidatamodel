=============
desidatamodel
=============

Introduction
============

This product defines the Data Model for DESI in the doc/ subdirectory.

Adding a new file
=================

When you add a new file, you also need to add it to a "toctree"
(Table of Contents Tree) of one of the index.rst files at that level or
higher.
