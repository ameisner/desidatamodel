==============
PROTODESI_DATA
==============

:envvar:`PROTODESI_DATA` contains raw data as produced by the telescope during
the ProtoDESI campaign.

The canonical location is ``$DESI_ROOT/protodesi/data`` (?), but one can set the
environment variable :envvar:`PROTODESI_DATA` to point anywhere.
