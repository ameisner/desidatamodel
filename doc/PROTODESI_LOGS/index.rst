==============
PROTODESI_LOGS
==============

:envvar:`PROTODESI_LOGS` contains output from tests conducted during the ProtoDESI campaign.
The canonical location is ``$DESI_ROOT/protodesi/logs`` (?), but one can set the
environment variable :envvar:`PROTODESI_DATA` to point anywhere.
