======
OBSCON
======

``OBSCON`` designates the observational conditions (or "layer") in which
targets will be observed. Possible values include "dark" and "bright" for
dark-time and bright-time targets, respectively. Under each observing
condition, targets are stored grouped by HEALPixel.
