==
HP
==

``HP`` is the HEALPixel number. Targets are stored in files that are split
by (nested) HEALPixel integer. For targets that are *not* resolved, the
``RESOLVE`` is omitted from the filename. For targets that are part of the
DESI Main Science Survey, ``PHASE`` is omitted from the filename.
