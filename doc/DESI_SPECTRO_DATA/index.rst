=================
DESI_SPECTRO_DATA
=================

$DESI_SPECTRO_DATA contains raw data as produced by the telescope.
The canonical location is $DESI_ROOT/spectro/data, but one can set the
environment variable $DESI_SPECTRO_DATA to point anywhere.
